**The Abandoned** is an action-adventure game in development at <ExtLink url="https://grumpusstudio.com/">The Grumpus Studio</ExtLink> using Unreal Engine 5, built by a team of industry veterans and skilled newcomers to the industry. I joined as a part-time Senior Software Engineer in February 2026, contributing to technical direction and implementation across multiple areas, including UI, player character movement, and <ExtLink url="https://dev.epicgames.com/documentation/en-us/unreal-engine/gameplay-ability-system-for-unreal-engine">GAS</ExtLink> integration.

The game is based on the <ExtLink url="https://www.goodreads.com/book/show/55276768-the-abandoned">novel of the same name</ExtLink>, written by Game Director Kol Humphries. Set in an overgrown, post-abandonment London, it follows Dani Wilson, a young American woman surviving in the city.

My contributions to the project include:

* Extensive implementation work with Common UI and the Gameplay Ability System
* Establishment of best practices for version control and development workflow
* Providing Unreal engineering expertise to the development team
