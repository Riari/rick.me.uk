Laravel Forum is a lean forum package for modern Laravel. I originally forked it from [github.com/atrakeur/laravel-forum](https://github.com/atrakeur/laravel-forum) to make some small improvements for my own needs, but I found myself getting carried away with it and ended up publishing it as a separate package.

It has since been through a few rewrites and I try to maintain it just enough to keep pace with major Laravel releases. Occasionally I develop new features for it, but it's intended to be a less opinionated option that leaves more of the specifics up to the user.

A demo is hosted at https://laravel-forum.teamteatime.net/ and the documentation is at https://www.teamteatime.net/docs/laravel-forum/.
