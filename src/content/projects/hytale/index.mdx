Hytale was a sandbox game under development at <ExternalLink url="https://hypixelstudios.com/">Hypixel Studios</ExternalLink>, where I worked as a senior software engineer. During my time in this role, I worked on various aspects of the game, including gameplay features, UI, and the technology behind Hytale's NPC behaviours.

It was built on a proprietery voxel engine developed in-house using <ExternalLink url="https://www.flecs.dev/flecs/">Flecs</ExternalLink>, a C/C++ ECS library. If you'd like to read more about the tech, you can find links to the official technical posts below.

* <ExternalLink url="https://web.archive.org/web/20250624000316/https://hytale.com/news/2024/12/technical-explainer-powering-up-with-launch-pads">December 2024 Technical Explainer: Powering Up with Launch Pads</ExternalLink>
* <ExternalLink url="https://web.archive.org/web/20250624000303/https://hytale.com/news/2024/6/summer-2024-technical-explainer-hytale-s-entity-component-system-oPwpCAMdI">Summer 2024 Technical Explainer: Hytale's Entity Component System</ExternalLink>

You can read more about the game on the <ExternalLink url="https://web.archive.org/web/20250624000305/https://www.hytale.com/news">official blog</ExternalLink>.

See below for a selection of in-engine videos and screenshots. The full set of Hytale media is also available on <ExternalLink url="https://archive.org/details/hytale-website-media-page-archive/">archive.org</ExternalLink>.
